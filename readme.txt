本项目只用于记录学习过程
文件说明：
main.py中的是整个python程序的入口，在环境中输入python main.py即可运行程序
build文件夹中的是用pyinstaller将main.py打包成exe文件的工程文件，可以不必理会
dist中的是用pyinstaller打包后生成的exe文件，可以直接运行，但是要注意确保工作目录为整个项目的根目录，否则会报错
src中的是项目的源代码
levels中的是项目的关卡文件
pic中的是项目的图片资源，所用的图片资源均来自开源网络资源，感谢dotown.maeda-design-room.net的图片资源

游戏说明：
注意游戏运行时要切换输入法为英文
确保工作目录是整个项目的根目录，然后在环境中输入python main.py即可运行游戏或者直接运行dist中的exe文件
通过按下左右键来切换关卡，按下's'即可使程序通过A*算法自动寻路，按下'g'即可使程序通过算法生成新的关卡
注意不要在程序运行时移动窗口或者切换后台，否则可能会导致程序崩溃，这是pygame的固有问题
